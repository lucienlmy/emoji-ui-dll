## 各区域详细说明

### 1. 标题栏区域
```
┌─────────────────────────────────────────────────────────────────────────────┐
│  🌐 多标签浏览器                                             ─  □  ×     │
├─────────────────────────────────────────────────────────────────────────────┤
```
- 使用 `创建Emoji窗口_字节集_扩展` 创建深色背景窗口
- 标题栏背景色：深灰 `RGB(45, 45, 48)`
- 客户区背景色：浅灰 `RGB(240, 240, 240)`
- 通过 `设置标题栏文字颜色` 设为白色
- 通过 `设置标题栏字体` 设为 "微软雅黑" 13号

### 2. 标签栏区域（TabControl）
```
  ┌──────────┐┌──────────┐┌──────────┐┌──────────┐              ┌────┐
  │ 🏠 首页  ×││ 📰 新闻  ×││ 📺 视频  ×││ ⚙ 设置  ×│   ...       │ ➕ │
  └──────────┘└──────────┘└──────────┘└──────────┘              └────┘
```
- TabControl 位于窗口顶部，紧贴标题栏下方
- 启用关闭按钮（`设置标签页可关闭`）→ 每个标签右侧显示 ×
- 启用拖拽排序（`设置标签页可拖拽`）→ 可拖动调整标签顺序
- 启用滚动模式（`设置标签栏可滚动`）→ 标签过多时单行滚动
- 标签页尺寸：`180 × 34`（宽一些，显示网页标题）
- 右侧放一个 ➕ 新建标签按钮（独立按钮，不属于TabControl）
- 配色方案：
  - 选中背景：白色 `ARGB(255, 255, 255, 255)`
  - 未选中背景：浅灰 `ARGB(255, 230, 230, 230)`
  - 选中文字：蓝色 `ARGB(255, 64, 158, 255)`
  - 未选中文字：深灰 `ARGB(255, 100, 100, 100)`
  - 指示条：蓝色 `ARGB(255, 64, 158, 255)`

### 3. 导航栏区域（每个Tab内容窗口的顶部）
```
  ┌────┬────┬────┬────┬──────────────────────────────────────┬────┐
  │ ←  │ →  │ 🔄 │ 🏠 │  🔒 https://www.example.com         │ ⭐ │
  └────┴────┴────┴────┴──────────────────────────────────────┴────┘
   后退  前进 刷新  主页              地址栏                    收藏
```
- 每个Tab的内容窗口内，顶部创建一行导航控件
- 后退/前进/刷新/主页：4个 Emoji 按钮（`创建Emoji按钮_字节集`）
  - ← 后退：`{ 226, 172, 133 }` (⬅)  或用文字 "←"
  - → 前进：`{ 226, 158, 161 }` (➡)  或用文字 "→"
  - 🔄 刷新：`{ 240, 159, 148, 132 }`
  - 🏠 主页：`{ 240, 159, 143, 160 }`
- 地址栏：编辑框（`创建编辑框`），单行、带边框、垂直居中
- 收藏按钮：⭐ `{ 226, 173, 144 }`
- 导航栏高度：约 40px
- 导航栏背景色：与Tab内容区域一致

### 4. 浏览器内容区域（CEF3 嵌入区域）
```
  ┌─────────────────────────────────────────────────────────────────────┐
  │                                                                     │
  │                                                                     │
  │                     CEF3 浏览器内容区域                              │
  │                   （每个Tab的内容窗口内嵌）                          │
  │                                                                     │
  │                                                                     │
  └─────────────────────────────────────────────────────────────────────┘
```
- 位于导航栏下方，填满Tab内容窗口的剩余空间
- 通过 `获取Tab内容窗口` 获取每个Tab的内容窗口句柄
- 在内容窗口中，使用 `CreateWindowExA("Static", ...)` 创建一个 Win32 Static 子窗口作为 CEF 容器
- 容器窗口样式：`WS_CHILD | WS_VISIBLE | WS_CLIPCHILDREN`（0x52000000）
- CEF3 嵌入方式：将容器句柄作为 `CefWindowInfo.SetAsChild()` 的父窗口
- 容器位置 = `(0, 导航栏高度, 内容窗口宽度, 内容窗口高度 - 导航栏高度)`

### 5. 状态栏区域
```
  ┌─────────────────────────────────────────────────────────────────────┐
  │  就绪                                              ▏ 100%          │
  └─────────────────────────────────────────────────────────────────────┘
```
- 位于窗口最底部（主窗口上，不在Tab内容窗口内）
- 左侧：状态标签（`创建标签`），显示加载状态/链接预览
- 右侧：进度条（`创建进度条`），显示页面加载进度
- 状态栏高度：约 24px
- 背景色：浅灰 `ARGB(255, 245, 245, 245)`

## 窗口尺寸与坐标规划

```
窗口总尺寸：1200 × 800（默认）

Y坐标分布：
  ┌─ Y=0    标签栏（TabControl）    高度=38
  ├─ Y=38   Tab内容区域开始
  │   ├─ 内容Y=0    导航栏           高度=44
  │   ├─ 内容Y=44   CEF3浏览器区域   高度=自适应
  │   └─ 填满剩余空间
  ├─ Y=窗口高-28  状态栏             高度=24
  └─ Y=窗口高     窗口底部
```

## 交互行为

| 操作 | 行为 |
|------|------|
| 点击 ➕ 按钮 | 新建标签页，默认打开主页 |
| 点击标签页 × | 关闭该标签页，关闭对应CEF浏览器实例 |
| 拖拽标签页 | 调整标签页顺序 |
| 双击标签页 | 可选：重命名标签页标题 |
| 右键标签页 | 弹出菜单：新建/关闭/关闭其他/关闭右侧 |
| Tab切换 | 切换显示对应的导航栏+CEF浏览器 |
| 地址栏回车 | 导航到输入的URL |
| 窗口大小改变 | TabControl和CEF浏览器区域自适应调整 |
| 页面加载中 | 状态栏显示"正在加载..."，进度条更新 |
| 页面加载完成 | 更新标签页标题为网页标题 |

## 关键回调

| 回调 | 用途 |
|------|------|
| `Tab切换回调` | 切换Tab时显示/隐藏对应的CEF浏览器 |
| `标签页关闭回调` | 关闭Tab时销毁CEF浏览器实例 |
| `标签页右键回调` | 弹出右键菜单 |
| `标签页双击回调` | 可选：重命名标签页 |
| `窗口大小改变回调` | 调整TabControl和内部控件尺寸 |
| `按钮点击回调` | 处理导航按钮和新建标签按钮 |
