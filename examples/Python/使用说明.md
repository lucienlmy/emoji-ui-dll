## ⚠️ 常见问题

### Q1: 提示找不到 DLL？

**解决方案：**
程序会自动搜索以下位置：
1. `emoji_window.dll`（当前目录）
2. `../../x64/Release/emoji_window.dll`（64位）
3. `../../Release/emoji_window.dll`（32位）

如果都找不到，请将 DLL 复制到当前目录。

### Q2: DLL 加载失败？

**可能原因：**
1. **Python 位数与 DLL 不匹配**
   ```cmd
   # 检查 Python 位数
   python -c "import struct; print(struct.calcsize('P') * 8, 'bit')"
   ```
   - 32位 Python → 使用 `Release/emoji_window.dll`
   - 64位 Python → 使用 `x64/Release/emoji_window.dll`

2. **缺少 Visual C++ 运行库**
   - 下载安装：https://aka.ms/vs/17/release/vc_redist.x64.exe

### Q3: 窗口显示后程序卡住？

这是**正常现象**！程序进入了消息循环，窗口正在显示。
- 关闭窗口即可退出程序
- 这不是卡死，而是在等待用户操作
