# Python 示例代码

✅ **已完成！**

本目录包含 emoji_window.dll 的 Python 调用示例，使用 `ctypes` 库实现。

## 📁 项目结构

```
Python/
├── emoji_window.py          # DLL 封装类
├── demo.py                  # 示例程序
├── run.bat                  # 运行脚本（Windows）
├── run.sh                   # 运行脚本（Linux/Mac）
├── requirements.txt         # 依赖列表
└── README.md               # 详细文档
```

## 🚀 快速开始

### 方法 1：使用运行脚本（推荐）

```cmd
# Windows
run.bat

# Linux/Mac
chmod +x run.sh
./run.sh
```

### 方法 2：直接运行

```cmd
python demo.py
```

## ✨ 功能展示

- ✅ 创建窗口（带 Unicode 表情）
- ✅ 3个分组框
- ✅ 多个标签（带 Emoji 图标）
- ✅ 多个按钮（带 Emoji 图标）
- ✅ 复选框（带回调）
- ✅ 信息框（带 Unicode 表情）

## ⚠️ 重要提示

**Python 位数必须与 DLL 位数匹配：**

```cmd
# 检查 Python 位数
python -c "import struct; print(struct.calcsize('P') * 8, 'bit')"

# 32位 Python → 使用 32位 DLL
# 64位 Python → 使用 64位 DLL
```
