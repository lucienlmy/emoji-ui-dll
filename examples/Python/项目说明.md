# Python 示例项目说明

## 📋 项目概述

这是一个完整的 Python 示例项目，展示如何使用 `ctypes` 库调用 emoji_window.dll，实现带有 Unicode 表情的 Windows 桌面应用程序。

## 🎯 设计目标

1. **简单易用**：封装 DLL 调用细节，提供 Pythonic API
2. **完整性**：覆盖主要控件（窗口、标签、按钮、分组框、复选框、信息框）
3. **可运行**：开箱即用，无需额外依赖
4. **跨平台脚本**：提供 Windows 和 Linux/Mac 运行脚本

## 🔧 技术要点

### 1. ctypes 基础

```python
import ctypes
from ctypes import c_int, c_uint, c_bool, c_void_p, c_char_p

# 加载 DLL
dll = ctypes.WinDLL("emoji_window.dll")

# 设置函数签名
dll.create_window_bytes.argtypes = [c_char_p, c_int, c_int, c_int]
dll.create_window_bytes.restype = c_void_p

# 调用函数
hwnd = dll.create_window_bytes(b"Title", 5, 800, 600)
```

### 2. 回调函数

```python
from ctypes import WINFUNCTYPE

# 定义回调类型（stdcall 调用约定）
ButtonClickCallback = WINFUNCTYPE(None, c_int, c_void_p)

# 创建回调函数
def on_button_click(button_id, parent_hwnd):
    print(f"按钮 {button_id} 被点击")

# 设置回调（必须保持引用！）
callback = ButtonClickCallback(on_button_click)
dll.set_button_click_callback(callback)
```

### 3. UTF-8 编码

```python
def str_to_utf8(text):
    """将 Python 字符串转换为 UTF-8 字节"""
    if isinstance(text, str):
        return text.encode('utf-8')
    return text

# 使用
title = "🎨 我的窗口"
title_bytes = str_to_utf8(title)
hwnd = dll.create_window_bytes(title_bytes, len(title_bytes), 800, 600)
```

### 4. 颜色值转换

```python
@staticmethod
def argb(a, r, g, b):
    """创建 ARGB 颜色值"""
    return (a << 24) | (r << 16) | (g << 8) | b

# 使用
color = EmojiWindow.argb(255, 64, 158, 255)  # #409EFF
```

## ⚠️ 常见陷阱

### 1. 回调被回收

```python
# ❌ 错误
def setup():
    callback = ButtonClickCallback(on_click)
    dll.set_button_click_callback(callback)
    # callback 离开作用域后会被回收！

# ✅ 正确
class App:
    def setup(self):
        self._callback = ButtonClickCallback(on_click)
        dll.set_button_click_callback(self._callback)
```

### 2. 位数不匹配

```python
# 检查 Python 位数
import struct
print(struct.calcsize('P') * 8, 'bit')

# 32位 Python 只能加载 32位 DLL
# 64位 Python 只能加载 64位 DLL
```

### 3. 字符串编码

```python
# ❌ 错误：直接传递字符串
dll.CreateLabel(..., "Hello", ...)

# ✅ 正确：转换为 UTF-8 字节
text = "Hello"
text_bytes = text.encode('utf-8')
dll.CreateLabel(..., text_bytes, len(text_bytes), ...)
```

### 4. 句柄类型

```python
# ctypes 中句柄使用 c_void_p
hwnd = dll.create_window_bytes(...)  # 返回 c_void_p

# 可以直接传递给其他函数
dll.CreateLabel(hwnd, ...)  # hwnd 作为 parent 参数
```

### 2. 事件系统

```python
class EventEmitter:
    def __init__(self):
        self._handlers = {}
    
    def on(self, event, handler):
        if event not in self._handlers:
            self._handlers[event] = []
        self._handlers[event].append(handler)
    
    def emit(self, event, *args):
        if event in self._handlers:
            for handler in self._handlers[event]:
                handler(*args)

class EmojiButton:
    def __init__(self):
        self.events = EventEmitter()
    
    def on_click(self, handler):
        self.events.on('click', handler)
```

class Config:

    @staticmethod

### 4. 异步操作

```python
import asyncio
import threading

class AsyncApp:
    async def load_data(self):
        """异步加载数据"""
        await asyncio.sleep(1)  # 模拟耗时操作
        return "数据加载完成"
    
    def on_button_click(self, btn_id, hwnd):
        """按钮点击处理"""
        # 在新线程中运行异步任务
        def run_async():
            loop = asyncio.new_event_loop()
            asyncio.set_event_loop(loop)
            result = loop.run_until_complete(self.load_data())
            print(result)
        
        thread = threading.Thread(target=run_async)
        thread.start()
```
