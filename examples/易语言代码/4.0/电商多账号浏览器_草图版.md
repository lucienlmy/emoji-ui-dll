# 电商多账号浏览器草图版

本示例对应当前 Python 原型：

- `examples/Python/电商多账号浏览器/demo_env_workspace_sketch.py`

更接近直接粘贴到 IDE 的骨架版见：

- [`电商多账号浏览器_草图版_可直接粘贴骨架.md`](./电商多账号浏览器_草图版_可直接粘贴骨架.md)

目标界面与结构：

- 左侧固定环境面板
- 顶部按钮区：`新建环境`、`删除环境`、`换肤`
- 树形菜单：
  - `环境切换`
    - `Amazon 店群`
    - `TikTok 店群`
    - `独立站`
  - `代理管理`
  - `分组管理`
  - `RPA自动化`
  - `插件中心`
  - `团队协作`
  - `系统设置`
- 右侧工作区：
  - 顶部环境信息栏
  - 操作工具栏
  - 最大化浏览器占位区

## 说明

这是按易语言风格整理的 `.md` 代码稿，便于直接照着实现。  
具体 DLL 命令名如果与你本地模块有差异，以仓库中的 `DLL命令.md` 和 4.0 文档为准。

另外有两个和 Python/C# 版一致的限制：

- `SetButtonRound` 只有开关，没有 3px/5px 半径参数
- 按钮坐标建议按主窗口绝对坐标来摆，不要假设一定是父面板局部坐标
