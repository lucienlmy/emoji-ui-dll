' 控件API（如设置标签位置、设置按钮位置等）接受的是逻辑像素。
' 在高DPI显示器上（如150%缩放），物理像素 = 逻辑像素 × 缩放比例。
' 例如：窗口创建1200x800逻辑像素，回调传入1392x928物理像素。
' 解决方案：用第一次回调的值和创建时的值计算缩放比例，后续按比例换算。
'
' 【标题栏高度说明】

' 状态栏是parent_drawn标签（D2D绘制），坐标是客户区绝对坐标，
' 所以状态栏Y = Tab区域高度 + 标题栏高度，刚好在TabControl底部下方。
