## 功能概述

本项目提供了 **非阻塞、macOS 扁平化风格** 的信息框组件，完美支持 UTF-8 和彩色 Emoji。

### 主要特性

✅ **非阻塞模式** - 不会卡死主窗口消息循环
✅ **macOS 扁平化设计** - 简洁、现代、优雅
✅ **完美 Emoji 支持** - 彩色表情符号显示
✅ **易语言友好** - 提供完整的易语言调用接口
✅ **硬件加速渲染** - 使用 Direct2D 和 DirectWrite

## 技术特性

### 非阻塞实现

**关键改进：**

```cpp
// ❌ 旧版本（阻塞）
void show_message_box_bytes(...) {
    HWND hwnd = CreateMessageBoxWindow(...);

    // 独立消息循环 - 会阻塞主窗口！
    MSG msg;
    while (GetMessage(&msg, hwnd, 0, 0)) {
        TranslateMessage(&msg);
        DispatchMessage(&msg);
    }
}

// ✅ 新版本（非阻塞）
void show_message_box_bytes(...) {
    HWND hwnd = CreateMessageBoxWindow(...);
    ShowWindow(hwnd, SW_SHOW);
    UpdateWindow(hwnd);

    // 不运行独立消息循环
    // 让主窗口的消息循环处理所有消息
}
```

**工作原理：**

1. 信息框窗口由主窗口的 `run_message_loop()` 统一处理
2. 点击按钮后触发 `CloseMessageBox()`
3. 如果有回调函数，在关闭窗口后调用
4. 主窗口始终保持响应

### Direct2D 渲染

**渲染流程：**

```cpp
void DrawMsgBox(ID2D1HwndRenderTarget* rt, ...) {
    rt->BeginDraw();

    // 1. 纯白背景 + 圆角
    rt->FillRoundedRectangle(bg_rect, white_brush);

    // 2. 极细边框
    rt->DrawRoundedRectangle(bg_rect, border_brush, 1.0f);

    // 3. 图标 Emoji (24px, 彩色)
    rt->DrawText(icon_emoji, ..., D2D1_DRAW_TEXT_OPTIONS_ENABLE_COLOR_FONT);

    // 4. 标题 (18px, 粗体)
    rt->DrawText(title, ...);

    // 5. 消息 (14px, 1.5倍行高)
    rt->DrawText(message, ...);

    // 6. 扁平按钮
    DrawButton(rt, factory, ok_button);
    DrawButton(rt, factory, cancel_button);

    rt->EndDraw();
}
```

## C++ 接口

### 1. 信息提示框（单按钮）

```cpp
void __stdcall show_message_box_bytes(
    HWND parent,                        // 父窗口句柄（可为 NULL）
    const unsigned char* title_bytes,   // 标题 UTF-8 字节集
    int title_len,                      // 标题字节长度
    const unsigned char* message_bytes, // 消息 UTF-8 字节集
    int message_len,                    // 消息字节长度
    const unsigned char* icon_bytes,    // 图标 Emoji UTF-8 字节集
    int icon_len                        // 图标字节长度
);
```

**特点：**
- 只有一个"确定"按钮
- 非阻塞，立即返回
- 点击按钮或按 Enter/Esc 关闭

### 2. 确认对话框（双按钮 + 回调）

```cpp
void __stdcall show_confirm_box_bytes(
    HWND parent,                        // 父窗口句柄（可为 NULL）
    const unsigned char* title_bytes,   // 标题 UTF-8 字节集
    int title_len,                      // 标题字节长度
    const unsigned char* message_bytes, // 消息 UTF-8 字节集
    int message_len,                    // 消息字节长度
    const unsigned char* icon_bytes,    // 图标 Emoji UTF-8 字节集
    int icon_len,                       // 图标字节长度
    MessageBoxCallback callback         // 回调函数指针
);
```

**回调函数原型：**

```cpp
typedef void (__stdcall *MessageBoxCallback)(int confirmed);
// confirmed: 1 = 用户点击确定, 0 = 用户点击取消
```

**特点：**
- 有"确定"和"取消"两个按钮
- 非阻塞，立即返回
- 用户选择后通过回调函数通知结果

## 常见问题

### Q1: 信息框会阻塞主窗口吗？

**A:** 不会！新版本采用非阻塞设计，信息框由主窗口的消息循环统一处理，不会卡死主窗口。

```易语言
' ✅ 正确：非阻塞
信息提示框_ (...)
调试输出 ("立即执行")  ' 这行代码会立即执行

' ❌ 错误理解：以为会等待用户点击
信息提示框_ (...)
' 不会在这里等待！函数已经返回了
```

### Q2: 如何获取用户的选择结果？

**A:** 使用确认框的回调函数：

```易语言
' 显示确认框
确认框_ (..., &我的回调函数)

' 回调函数（必须声明为"公开"）
.子程序 我的回调函数, , 公开
.参数 确认, 整数型

.如果 (确认 ＝ 1)
    ' 用户点击了确定
.否则
    ' 用户点击了取消
.如果结束
```

### Q3: 如何显示多行消息？

**A:** 在消息字节集中使用 `\n` (字节 10)：


### Q5: 如何自定义按钮颜色？

**A:** 当前版本使用固定的 macOS 风格颜色：
- 主按钮：`#007AFF` (蓝色)
- 次要按钮：`#F2F2F7` (浅灰)

如需自定义，需要修改 C++ 源码中的 `CreateMessageBoxWindow()` 函数。

### Q6: 信息框的父窗口参数有什么用？

**A:** 父窗口参数用于：
1. **居中显示**：信息框会在父窗口中央显示
2. **模态行为**：显示信息框时，父窗口会被禁用（`EnableWindow(parent, FALSE)`）
3. **焦点管理**：关闭信息框后，焦点返回父窗口

```易语言
' 有父窗口（推荐）
信息提示框_ (窗口句柄, ...)  ' 在主窗口中央显示

' 无父窗口
信息提示框_ (0, ...)  ' 在屏幕中央显示
```

### Q7: 编译 DLL 需要什么环境？

**A:** 需要：
- Visual Studio 2019 或更高版本
- Windows SDK 10.0 或更高版本
- 编译为 32 位 (x86) 版本

```bash
# 命令行编译
msbuild emoji_window.sln /p:Configuration=Release /p:Platform=Win32
```

### Q8: 如何调试信息框？

**A:** 使用 Visual Studio 附加到易语言进程：

1. 在 C++ 代码中设置断点
2. 运行易语言程序
3. Visual Studio → 调试 → 附加到进程
4. 选择易语言进程
5. 触发信息框，断点会命中
