# 易语言编码规则

## 全角符号规则

易语言中赋值、运算、比较等符号必须使用全角字符：

| 用途 | 错误（半角） | 正确（全角） |
|------|------------|------------|
| 赋值/等于 | `=` | `＝` |
| 加法/拼接 | `+` | `＋` |
| 大于 | `>` | `＞` |
| 小于 | `<` | `＜` |
| 大于等于 | `>=` | `` |
| 小于等于 | `<=` | `` |
| 不等于 | `!=` | `` |

## 禁止使用的命令

| 错误命令 | 正确命令 |
|---------|---------|
| `取字节集指针` | `取变量数据地址` |
| `取字节集数据地址` | `取变量数据地址` |
| `位左移` | `左移` |

## 按钮ID必须保存到程序集变量

`创建Emoji按钮_字节集` 返回的按钮ID是动态分配的，禁止用硬编码数字判断。

```
' ✅ 正确
.程序集变量 按钮_获取文本, 整数型
按钮_获取文本 ＝ 创建Emoji按钮_字节集 (主窗口句柄, ...)

' ❌ 错误 - 硬编码数字
.判断开始 (按钮ID ＝ 1)
```

## 按钮点击回调签名

回调必须有2个参数：

```
.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 _父窗口句柄, 整数型
```

## UTF-8 字节集规则

DLL返回的UTF-8字节集，禁止用 `到文本()` 或 `编码_Utf8到Ansi()` 转换。必须全程用字节集拼接。

```
' ✅ 正确 - 全程字节集拼接
.局部变量 结果字节集, 字节集
结果字节集 ＝ 编码_Ansi到Utf8 ("文本内容：") ＋ 文本缓冲区 ＋ { 10 } ＋ 编码_Ansi到Utf8 ("字节长度：") ＋ 编码_Ansi到Utf8 (到文本 (文本长度))
设置标签文本 (标签_结果, 取变量数据地址 (结果字节集), 取字节集长度 (结果字节集))

' ❌ 错误 - 到文本() 会导致乱码
文本内容 ＝ 到文本 (文本缓冲区)
```

换行符用 `{ 10 }`，不用 `#换行符`。

## Emoji 字节集硬编码

易语言IDE使用ANSI编码，不支持Unicode。Emoji必须用UTF-8字节集硬编码：

```
' ✅ 正确
utf8字节集 ＝ { 240, 159, 142, 168, 32 } ＋ 编码_Ansi到Utf8 ("自定义主题")

' ❌ 错误 - emoji会变问号
utf8字节集 ＝ 编码_Ansi到Utf8 ("🎨 自定义主题")
```

## 窗口标题用字节集版本

```
' ✅ 正确
utf8字节集 ＝ 编码_Ansi到Utf8 ("主题切换示例 ") ＋ { 240, 159, 142, 168 }
窗口句柄 ＝ 创建Emoji窗口_字节集 (取变量数据地址 (utf8字节集), 取字节集长度 (utf8字节集), 600, 500)

' ❌ 错误
窗口句柄 ＝ 创建Emoji窗口 ("主题切换示例 🎨", 600, 500)
```

## 条件判断语法

- `.如果` / `.如果结束`  支持 `.否则` 分支
- `.如果真` / `.如果真结束`  不支持 `.否则`

需要 `.否则` 时必须用 `.如果`。
