# Emoji Window 火山界面库

基于 `emoji_window.dll`（Direct2D 渲染）的火山 PC 界面组件库，支持无边框自定义窗口、emoji 彩色文字、深色/浅色主题。

## 快速开始

### 1. 安装

1. 将整个 `voldev` 目录放到火山平台类库目录下（如 `火山平台\plugins\vprj_win\classlib\sys\emoji_window`）
2. 重启火山平台
3. 在工具箱中可以看到 **Emoji Window** 分类

### 2. 添加依赖

新建火山 PC 项目后，在项目属性中添加引用：
- 右键项目 → 添加引用 → 找到 `emoji_window_ui` 并勾选

### 3. DLL 部署

编译后需将对应的 `emoji_window.dll` 放到 EXE 同目录：

| 目标平台 | DLL 文件 |
|----------|---------|
| 64 位 | `runtime_x64/emoji_window.dll` |
| 32 位 | `runtime_win32/emoji_window.dll` |

## 可用组件

| 组件 | 说明 | 工具箱图标 |
|------|------|-----------|
| **Emoji顶层窗口** | 自定义无边框主窗口，支持标题栏 emoji 文字、深色/浅色主题 | `res/emoji_top_window.bmp` |
| **Emoji面板** | 容器控件，可容纳其他子控件，支持独立背景色 | `res/emoji_panel.bmp` |
| **Emoji按钮** | 支持 emoji 图标的按钮，圆角、悬停/按下颜色可定制 | `res/emoji_button.bmp` |
| **Emoji标签** | 显示文本（含 emoji），字体大小/颜色/对齐可定制 | `res/emoji_label.bmp` |
| **Emoji编辑框** | 单行/多行文本输入框，密码模式、只读模式 | `res/emoji_edit.bmp` |

## 组件详细用法

### Emoji顶层窗口

基础属性：

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 标题 | 文本 | "Emoji窗口" | 窗口标题，支持 emoji（UTF-8 字节） |
| 背景颜色 | 整数 | `0x40EFEF` | 客户端背景色（ARGB 格式，如 `0xFF232C3F`） |
| 边框颜色 | 整数 | -1 | 窗口边框色，-1 跟随主题 |
| 标题栏对齐 | 文本 | "居中" | 标题文字对齐："靠左"、"居中"、"靠右" |
| 标题栏字体大小 | 小数 | 12.0 | 标题栏文字大小 |
| 标题栏颜色 | 整数 | 0 | 0=跟随主题，非0=自定义 ARGB |
| 显示模式 | 整数 | 0 | 0=正常窗口，可拖大小；其它值待扩展 |

**创建窗口示例（火山代码）：**

```
// 1. 从工具箱拖一个 Emoji顶层窗口 到设计器
// 2. 设置属性：
//    标题 = "😀 我的应用"
//    背景颜色 = 0xFF1E1E2E
//    标题栏对齐 = "居中"
// 3. 运行即可看到无边框窗口
```

### Emoji面板

用作容器，放置子控件。面板有自己的背景色，子控件定位相对于面板左上角。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 背景颜色 | 整数 | -1 | -1=透明/跟随父窗口 |

**使用方法：**
```
// 1. 在窗口上放一个 Emoji面板
// 2. 设置面板的位置和大小
// 3. 在面板内放置按钮、标签等子控件
// 面板会自动裁剪子控件，超出部分不显示
```

## 主题切换

所有组件支持浅色/深色主题自动切换：

```
// 切换主题
EmojiWindow.置主题模式(1)  // 0=浅色，1=深色
```

主题会自动应用到该窗口和所有子控件。背景颜色设为 0 的控件会跟随主题变化。

## 颜色格式说明

所有颜色属性使用 ARGB 格式（16 进制整数）：

```
0xAARRGGBB

示例：
  0xFF232C3F  → 深灰蓝（不透明）
  0xFFFFFFFF  → 白色
  0xFF121212  → 接近黑色
  0x40FFFFFF  → 半透明白色
  -1           → 跟随主题
   0           → 跟随主题（标题栏/文字颜色）
```

## 布局

支持在窗口上使用**布局管理器**自动排列子控件：

1. 在窗口属性中设置布局类型（流式/网格/停靠）
2. 设置间距、内边距
3. 子控件按声明顺序自动排列
4. 窗口大小变化时自动重排

布局相关的方法由 `EmojiWindow` 类提供（`置布局类型`、`置布局间距` 等）。

## 与 易语言 的关系

本库底层与 `易语言` 示例目录下的 DLL 为同一份 `emoji_window.dll`。火山版通过 `EmojiWindowLib` 类封装了所有 C 导出函数，提供面向对象的接口。`src/emoji_window_ui_接口.wsv` 中声明了所有底层 API 映射。

## 常见问题

### Q: 运行后窗口是空白的？
A: 检查 `emoji_window.dll` 是否放在 EXE 同目录。32 位 EXE 需要 32 位 DLL，64 位 EXE 需要 64 位 DLL。

### Q: Emoji 文字显示为问号？
A: 火山/易语言 IDE 使用 ANSI 编码，不支持直接输入 emoji。需要将 emoji 转换为 UTF-8 字节数组再设置文本属性。

### Q: 如何获取子控件的句柄？
A: 组件对象有 `取窗口句柄()` 或 `取容器句柄()` 方法，返回 Windows HWND 用于高级操作。

### Q: 控件拖放时闪烁？
A: 这是 D2D 和 GDI 子窗口混合渲染的已知问题。闪烁原理及修复进度见主仓库的 commit 记录和 `emoji_window.cpp` 中 `WM_ENTERSIZEMOVE`/`WM_EXITSIZEMOVE` 的实现。
