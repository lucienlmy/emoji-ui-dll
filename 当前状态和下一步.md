# Tab切换问题 - 当前状态和下一步

## 当前情况

你遇到的问题：
1. ❌ 切换到Tab2时，Tab1的按钮仍然显示
2. ❌ Tab2中的编辑框不显示
3. ✅ 已经尝试在易语言层面手动调用 `ShowWindow` 隐藏Tab1，但无效

## 问题根源分析

这说明问题**不在窗口的可见性**，而在于：

### 最可能的原因：Direct2D绘制逻辑

按钮是通过Direct2D在 `WindowProc` 的 `WM_PAINT` 消息中绘制的。即使窗口被隐藏（`ShowWindow(hwnd, SW_HIDE)`），`WM_PAINT` 消息仍然可能被处理，导致按钮被绘制出来。

**关键代码位置**（emoji_window.cpp）：
```cpp
case WM_PAINT: {
    if (state) {
        PAINTSTRUCT ps;
        BeginPaint(hwnd, &ps);

        if (state->render_target) {
            state->render_target->BeginDraw();
            state->render_target->Clear(D2D1::ColorF(D2D1::ColorF::White));

            // 这里绘制所有按钮，没有检查窗口是否可见！
            for (const auto& button : state->buttons) {
                DrawButton(state->render_target, state->dwrite_factory, button);
            }

            state->render_target->EndDraw();
        }

        EndPaint(hwnd, &ps);
    }
    return 0;
}
```

## 必须修改DLL的原因

易语言层面的 `ShowWindow` 只能控制窗口的可见性标志，但不能阻止Direct2D的绘制。必须在DLL的 `WindowProc` 中添加可见性检查。

## 解决方案

### 方案1：修改WindowProc的WM_PAINT处理（推荐）

在 `emoji_window.cpp` 的 `WindowProc` 函数中，修改 `WM_PAINT` 处理：

```cpp
case WM_PAINT: {
    if (state) {
        PAINTSTRUCT ps;
        BeginPaint(hwnd, &ps);

        // ========== 添加可见性检查 ==========
        if (IsWindowVisible(hwnd)) {
            if (state->render_target) {
                state->render_target->BeginDraw();
                state->render_target->Clear(D2D1::ColorF(D2D1::ColorF::White));

                for (const auto& button : state->buttons) {
                    DrawButton(state->render_target, state->dwrite_factory, button);
                }

                state->render_target->EndDraw();
            }
        }
        // ========================================

        EndPaint(hwnd, &ps);
    }
    return 0;
}
```

### 方案2：修改UpdateTabLayout函数

同时改进 `UpdateTabLayout` 函数，使用更可靠的隐藏方法：

```cpp
void UpdateTabLayout(TabControlState* state) {
    if (!state || !state->hTabControl) return;

    // ... 获取显示区域代码 ...

    // 第一步：先隐藏所有非当前Tab
    for (size_t i = 0; i < state->pages.size(); i++) {
        if ((int)i != state->currentIndex) {
            TabPageInfo& page = state->pages[i];
            if (page.hContentWindow && IsWindow(page.hContentWindow)) {
                // 使用ShowWindow而不是SetWindowPos
                ShowWindow(page.hContentWindow, SW_HIDE);
                page.visible = false;
                
                // 强制触发WM_PAINT，让窗口知道自己被隐藏了
                InvalidateRect(page.hContentWindow, nullptr, TRUE);
            }
        }
    }

    // 第二步：显示当前Tab
    if (state->currentIndex >= 0 && state->currentIndex < (int)state->pages.size()) {
        TabPageInfo& page = state->pages[state->currentIndex];
        if (page.hContentWindow && IsWindow(page.hContentWindow)) {
            // 先设置位置和大小
            SetWindowPos(page.hContentWindow, HWND_TOP, 
                        rcTab.left, rcTab.top, contentWidth, contentHeight,
                        SWP_NOACTIVATE);
            
            // 再显示窗口
            ShowWindow(page.hContentWindow, SW_SHOW);
            page.visible = true;

            // 更新渲染目标...
            
            // 强制刷新
            InvalidateRect(page.hContentWindow, nullptr, TRUE);
            UpdateWindow(page.hContentWindow);
        }
    }
}
```
