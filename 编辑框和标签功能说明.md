## 🔧 编译步骤

### 方法一：使用 Visual Studio（推荐）

1. 打开 `emoji_window.sln` 解决方案
2. 选择 `Release` 配置和 `Win32` 平台
3. 点击"生成" → "生成解决方案"
4. 编译完成后，DLL 位于 `Win32\Release\emoji_window.dll`

### 方法二：使用命令行

在 **Developer Command Prompt for VS** 中运行：

```cmd
build_32bit.bat
```

### 方法三：手动编译

```cmd
cd emoji_window
cl /LD /EHsc /O2 /MD emoji_window.cpp /link /DEF:emoji_window.def ^
   d2d1.lib dwrite.lib comctl32.lib uxtheme.lib user32.lib gdi32.lib ^
   /OUT:../emoji_window.dll
```

## 🎨 颜色格式说明

颜色使用 ARGB 格式（32位整数）：

```
格式：0xAARRGGBB
- AA: Alpha（透明度）00-FF
- RR: Red（红色）00-FF
- GG: Green（绿色）00-FF
- BB: Blue（蓝色）00-FF
```

### 常用颜色值

| 颜色 | 十进制值 | 十六进制 | 说明 |
|------|---------|---------|------|
| 黑色 | 4278190080 | 0xFF000000 | 不透明黑色 |
| 白色 | 4294967295 | 0xFFFFFFFF | 不透明白色 |
| 红色 | 4294901760 | 0xFFFF0000 | 不透明红色 |
| 绿色 | 4278255360 | 0xFF00FF00 | 不透明绿色 |
| 蓝色 | 4278190335 | 0xFF0000FF | 不透明蓝色 |
| 黄色 | 4294967040 | 0xFFFFFF00 | 不透明黄色 |
| 浅黄色 | 4294951175 | 0xFFFFFF87 | 浅黄色背景 |
| 透明 | 0 | 0x00000000 | 完全透明 |

## ⚠️ 注意事项

1. **文本编码**：所有文本必须使用 UTF-8 编码的字节集
2. **字体名称**：推荐使用"微软雅黑"、"宋体"等系统字体
3. **颜色格式**：使用 ARGB 格式，注意透明度通道
4. **内存管理**：获取文本时需要预先分配足够的缓冲区
5. **编码转换**：需要使用 `编码_Ansi到Utf8` 和 `编码_Utf8到Ansi` 进行转换

## 🐛 常见问题

### Q1: 编译失败？
**A**: 确保安装了 Visual Studio 2019/2022 和 Windows 10 SDK

### Q2: 中文显示乱码？
**A**: 确保使用 UTF-8 编码，使用 `编码_Ansi到Utf8` 转换

### Q3: 颜色不正确？
**A**: 检查颜色格式是否为 ARGB（0xAARRGGBB）

### Q4: 获取文本失败？
**A**: 确保缓冲区足够大（建议 1024 字节以上）
